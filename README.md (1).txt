# Face Recognition Attendance System

## Objective
Automate attendance using facial recognition.

## Tools Used
- Python
- OpenCV
- Face Recognition
- NumPy
- Pandas
- Streamlit

## Project Workflow
1. Collect face images.
2. Preprocess images.
3. Detect faces.
4. Extract facial features.
5. Train recognition model.
6. Recognize faces.
7. Mark attendance.
8. Display attendance dashboard.

## Project Structure

FaceRecognitionAttendance/

- app.py
- train_model.py
- dashboard.py
- requirements.txt
- README.md
- attendance.csv
- dataset/

## Outcome
The system automatically records attendance and stores it for analysis.
